# India_vs_Germany_job_comparison
This project is an interactive Tableau visualization that compares job postings between India and Germany using a stacked bar chart. The analysis focuses on selected job roles and applies multiple filtering conditions to identify hiring patterns across the two countries.


Features

India vs Germany job posting comparison
Stacked bar chart visualization
Analysis of Data Scientist, Art Teacher, and Aerospace Engineer roles
Qualification-based filtering
Experience and salary-based filtering
Country-wise job trend comparison
Interactive Tableau visualization


Tools Used

Tableau Public
CSV Dataset
Stacked Bar Chart
Calculated Fields and Filters


Dashboard Insights

The visualization provides insights into:

Distribution of job postings across India and Germany
Comparison of selected job roles between countries
Hiring trends for Full-Time positions
Impact of experience and salary filters on job availability
Country-wise representation of job opportunities


Dataset Filters Used

Qualification = B.Tech
Work Type = Full-Time
Experience > 2 Years
Salary > 10000
Job Portal = Indeed
Country = India and Germany
Job Title = Data Scientist, Art Teacher, Aerospace Engineer
Company Name Length > 8 Characters
Location Not Empty


Dataset Source

The dataset used in this project is a self-created CSV dataset prepared for analytical and educational purposes.


Project Purpose

The main goal of this project is to compare job opportunities between India and Germany and identify differences in job posting distribution for selected professional roles using Tableau visualization.
